Deklaratywność

React sprawia, że bezbolesne jest tworzenie interaktywnych interfejsów użytkownika. Zaprojektuj proste widoki dla każdego stanu w aplikacji, a React skutecznie aktualizuje i wyświetla właściwe składniki po zmianie danych.

Widoki deklaratywne sprawiają, że Twój kod jest bardziej przewidywalny i łatwiejszy do debugowania.

Komponentowość

Zbuduj izolowane elementy zarządzające własnym stanem a następnie zbuduj z nich złożone interfejsy użytkownika.

Ponieważ logika komponentów jest zapisywana w kodzie JavaScript zamiast w szablonach, możesz łatwo przekazywać złożone dane za pośrednictwem aplikacji i trzymać stan poza strukturą DOM.

Naucz się raz, wykorzystuj wszędzie

Nie przyjmujemy założeń dotyczących reszty stosu technologicznego, dzięki czemu można rozwijać nowe funkcje w React bez konieczności przepisywania istniejącego kodu.

React może być renderowany na serwerze za pomocą Node i jako aplikacja mobilna za pomocą React Native.

Prosty komponent

Komponenty React implementują metodę render() która pobiera dane wejściowe i zwraca wynik wyjściowy do wyświetlenia. Ten przykład używa składni XML nazywanej JSX. Dane wejściowe, które są przekazane do komponentu mogą być dostępne przez metodę render() za pomocą this.props.

JSX is opcjonalny i nie jest wymagany do używania frameworka React. Wypróbuj klikając na zakładkę "Compiled JS" żeby zobaczyć czysty kod JavaScript utworzony przez kompilator JSX.

Komponent stanowy

Oprócz pobierania danych wejściowych (dostępnych za pomocą this.props), komponent może utrzymywać wewnętrzny stan danych (dostępnych za pomocą this.state). Gdy dane stanowe komponentu zmienią się, wyrenderowane elementy zostaną zaktualizowane poprzez ponowne wywołanie metody render().

Aplikacja

Używając własności props i state możesz utworzyć małą aplikację ToDo. Ten przykład używa własności state do przechowania listy z tekstem wprowadzonym przez użytkownika.

Komponent używający zewnętrznych wtyczek

React jest elastyczny i zapewnia haki (ang. hooks), które umożliwiają współpracę z innymi bibliotekami i frameworkami. Ten przykład używa Remarkable, zewnętrznej biblioteki Markdown, do konwersji wartości z pola textarea w czasie rzeczywistym.
